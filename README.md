# liri-node-app
Welcome to LIRI-CLI!!!

LIRI is a Language Interpretation and Recognition Interface for your computer command line.

Currently LIRI has 4 built in methods:

1. concert-this
2. spotify-this-song
3. movie-this
4. do-what-it-says
